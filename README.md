
# Hyper Bar ordering app

Refreshing app made for users to order drinks at a fictional "Hyper Bar"



## Highlights

- Firebase used for customer-bartender communication
- MVVM architecture
- real-time order functionality


## Tech Stack

Kotlin, Jetpack Compose
